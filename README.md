# Mod Repository

This repository contains **direct download links for Minecraft mods** organized by category. It is intended for easy access and quick setup of modpacks.

All mods listed here are provided as **direct Modrinth CDN links** for fast and reliable downloads.

## üõ† How to Add New Mods

Follow these steps carefully when adding new mods:

1. **Add the mod to the correct category**

   * If the category does not exist, create it.
2. **Use this exact format for the mod name in a comment:**
   `MOD NAME - ARCHITECTURE (FORGE / NEOFORGE) - VERSION`
3. **Add the direct download link from Modrinth below the comment**

   * The link must be the *direct file download*.
   * To get it: right-click the download button on Modrinth ‚Üí *Copy link*.
